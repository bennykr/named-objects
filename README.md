# named-objects
Header-only named object base class.

-------------------

Compilation and execution of tests with:
> mkdir build \
> cd build \
> cmake .. \
> make

Execution with:
> ./run_tests

A somewhat faulty (not taking into account unused inline functions) HTML coverage report with 'gcovr':
> make run_tests_coverage
